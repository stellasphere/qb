Question

Consider the function  f ( x ) = x 4 − 6 x 2 − 2 x + 4 ,  x ∈ R .

The graph of f is translated two units to the left to form the function g ( x ) .

Express  g ( x )  in the form  a x 4 + b x 3 + c x 2 + d x + e where  a ,  b ,  c ,  d ,  e ∈ Z .

Markscheme

* This question is from an exam for a previous syllabus, and may contain minor differences in marking or structure.

g ( x ) = f ( x + 2 ) ( = ( x + 2 ) 4 − 6 ( x + 2 ) 2 − 2 ( x + 2 ) + 4 )       M1

attempt to expand  ( x + 2 ) 4       M1

( x + 2 ) 4 = x 4 + 4 ( 2 x 3 ) + 6 ( 2 2 x 2 ) + 4 ( 2 3 x ) + 2 4        (A1)

= x 4 + 8 x 3 + 24 x 2 + 32 x + 16       A1

g ( x ) = x 4 + 8 x 3 + 24 x 2 + 32 x + 16 − 6 ( x 2 + 4 x + 4 ) − 2 x − 4 + 4

= x 4 + 8 x 3 + 18 x 2 + 6 x − 8       A1

Note: For correct expansion of  f ( x − 2 ) = x 4 − 8 x 3 + 18 x 2 − 10 x  award max  M0M1(A1)A0A1.
