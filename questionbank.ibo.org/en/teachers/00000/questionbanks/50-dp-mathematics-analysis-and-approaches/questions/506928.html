Question

Let  f ( x ) = 4 cos ( x 2 ) + 1 , for  0 ⩽ x ⩽ 6 π . Find the values of x for which f ( x ) > 2 2 + 1 .

Markscheme

correct working          (A1)

eg    4cosx2=22,  cosx2>22

recognizing   co s − 1 2 2 = π 4           (A1)

three correct values for x         A1A1

eg     π 2 ,  7 π 2 ,  9 π 2

valid approach to find intervals          (M1)

correct intervals (must be in radians)        A1A1     N2

Note: If working shown, award A1A0 if inclusion/exclusion of endpoints is incorrect. If no working shown award N1.
