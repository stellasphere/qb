Question

Find the acute angle between the planes with equations x + y + z = 3 and 2 x − z = 2 .

Markscheme

* This question is from an exam for a previous syllabus, and may contain minor differences in marking or structure.

n 1 = ( 1 1 1 ) and n 2 = ( 2 0 − 1 )      (A1)(A1)

EITHER

θ = arccos ⁡ ( n 1 ∙ n 2 | n 1 | | n 2 | ) ( cos ⁡ θ = n 1 ∙ n 2 | n 1 | | n 2 | )    (M1)

= arccos ⁡ ( 2 + 0 − 1 3 5 ) ( cos ⁡ θ = 2 + 0 − 1 3 5 )    (A1)

= arccos ⁡ ( 1 15 ) ( cos ⁡ θ = 1 15 )

OR

θ = arcsin ⁡ ( | n 1 × n 2 | | n 1 | | n 2 | ) ( sin ⁡ θ = | n 1 × n 2 | | n 1 | | n 2 | )    (M1)

= arcsin ⁡ ( 14 3 5 ) ( sin ⁡ θ = 14 3 5 )    (A1)

= arcsin ⁡ ( 14 15 ) ( sin ⁡ θ = 14 15 )

 

THEN

= 75.0 ∘  (or 1.31)    A1
